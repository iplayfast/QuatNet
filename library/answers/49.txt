A good list interface typically needs:
- add/append: add element to end
- insert: insert at index
- remove/delete: remove by index or value
- get: access element by index
- set: modify element at index
- size/length: number of elements
- contains/member: check if element exists
- iterate: support for-each traversal
- clear: remove all elements
- index_of: find position of element
- sublist/slice: get portion of list

Python's list implements most via __getitem__, __setitem__, append, insert, remove, pop, extend, etc.
Java's List interface adds iterator(), listIterator(), toArray().
C++ std::vector adds capacity management (reserve, shrink_to_fit).
