```python
def count_odd_numbers(lst):
    """
    Count the number of odd integers in a given list.
    
    Parameters:
        lst (list[int]): List of integers
    
    Returns:
        int: Count of odd numbers
    """
    return sum(1 for x in lst if x % 2 != 0)
```

### Example Usage:
```python
if __name__ == "__main__":
    numbers = [1, 2, 4, 6, 7, 9, 11]
    result = count_odd_numbers(numbers)
    print(f"Number of odd numbers in {numbers}: {result}")  # Output: 3
```
