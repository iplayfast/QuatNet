Here is a simple Python function that just squares any number passed into it:

```python
def square(n):
    return n**2

print(square(4))
```

This function takes an input `n`, and returns the value of `n` raised to the power of 2, which is the same as squaring `n`. Then we call our function with an argument of 4 to check if it works.

The output would be: `16`
