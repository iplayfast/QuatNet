"The Battleship game implementation features a 10x10 grid where ships are randomly placed with random orientations. On each turn, the program requests coordinates from the human opponent, reports hits or misses, and maintains a probability map for its own shots using a density-based targeting strategy. After a hit, it switches to a hunt mode that systematically explores adjacent cells in a cross pattern until the ship is sunk, then returns to the density search pattern. The game tracks hits, misses, and sunk ships for both players."
